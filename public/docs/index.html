本地预览

将 HTTP 服务器的文档根目录指向本仓库的 public 文件夹。示例（Python 内置服务器）：

cd public && python3 -m http.server 8080

在浏览器打开 http://127.0.0.1:8080/。所有页面使用以 / 开头的资源路径，因此需要 Web 服务器，而不是直接双击打开 HTML 文件。

工具路由生成

大量扁平工具路径（例如 /seedance/、/seedance-2-0/、/text-to-video/）由脚本生成，便于与常见聚合站的信息架构对齐，同时保持文案与样式原创。

在项目根目录执行：

python3 scripts/generate_pages.py

脚本会写入 public/<slug>/index.html 并刷新 public/sitemap.xml。手工维护的页面（如 public/blog/）不会被覆盖。

子域映射

法律与道德上不应复制第三方站点；本仓库提供的是 Sota Video Gen（品牌词：sota video gen）的原创页面。若你希望「主站 / 应用 / 博客 / 文档」分别用子域展示，可采用以下常见做法：

  • 同一静态目录 + 多主机名：在 CDN 或反向代理（如 Nginx、Cloudflare Workers）上为 app.example.com、blog.example.com 配置独立源站路径，分别映射到 /app/、/blog/、/docs/，并把该路径设为文档根或添加重写规则。
  • 多仓库或多构建产物：每个子域部署独立项目，仅共享设计系统；适合应用与营销站技术栈不同的情况。

本站页面链接已按路径划分：/ 营销首页、/ai-tools/ 工具目录、各工具子路径（如 /image-to-video/）、以及 /app/、/blog/、/docs/，便于你对照配置 DNS 与路由。
